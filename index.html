HTML & CSS

HTML

  1. HTML is an acronym for Hyper-Text Mark-up Language.
  2. Hypertext is the text present in the internet.
  3. Markup language is a way that computers to each other to control how text is processed and presented.
  4. HTML documents has a series of elements that explains the structure of the web page.
  5. HTML element contains a start tag, content and the end tag.
  6. These elements inform the browser on how to display the contents of the html document.

HTML Element:

<h1> Content <\h1>

In the above element <h1> is the start tag, and <\h1> is the end tag and within these tags is the content to be displayed.

Let's discuss few basic elements in HTML.

Heading tag:

There are six levels of heading defined by HTML.
They are <h1>...<\h1>,<h2>...<\h2>,<h3>...<\h3>,<h4>...<\h4>,<h5>...<\h5>,<h6>...<\h6>.
h1 is the highest level and h6 is the least level.
